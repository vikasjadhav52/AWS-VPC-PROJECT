# AWS VPC Project

## Project Overview
This project demonstrates creation of a custom AWS VPC infrastructure with secure networking components.

## AWS Services Used
- VPC
- EC2
- Internet Gateway
- NAT Gateway
- Route Tables
- Security Groups

## Architecture
Public and Private subnet architecture deployed successfully.

## Steps Performed
1. Created custom VPC
2. Created public subnet
3. Created private subnet
4. Attached Internet Gateway
5. Configured Route Tables
6. Created Bastion Host
7. Connected private server using Bastion Host

## Outcome
Successfully deployed secure AWS network architecture with controlled internet access.

## Skills Demonstrated
- AWS Networking
- Linux
- SSH Connectivity
- Cloud Security
- Infrastructure Setup
